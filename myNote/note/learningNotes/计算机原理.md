# 计算机原理

## 运算(operation)

### 异或

本质上为找出差异的过程，找出所有差异时，异或运算即完成。这是一个从旧信息产出新信息的过程。

由此具有以下性质：

- 任何数和 0 做异或运算，结果仍然是原来的数；a（xor）0=a
- 任何数和其自身做异或运算，结果是 0；a（xor）a=0
- 满足交换律和结合律；



异或运算可运用于去除冗余信息，数据压缩等方面，自然界中也随处可见异或运算的运用。

例：

利用异或交换a，b两个变量数据：

```javascript
let a=1;b=2;
a=a^b;
b=a^b;
a=a^b;//a=2,b=1
//js中可以使用更简便的解构[a,b]=[b,a]
```
